# 飞鼠四方支付系统

使用go(golang)语言编写的四方支付系统

### 特点

* 编译成**二进制文件**，彻底避免**源码泄漏**。
* 支持**缓存**、**消息队列**、**异步任务**、**定时任务**。
* **实时数据分析**
* 并发高。4核8G服务器（RMB200元左右），日处理订单**一百万+**
